# Waliworks Website

A one-page Bun + Vite + React + Tailwind CSS website for Waliworks, a micro motorcycle workshop led by Joshua Acosta and Lian Reyes.

## What is included

- Sticky responsive header with hamburger navigation
- Dark motorcycle garage styling with orange and yellow accents
- Hero, services, work gallery, before-and-after cards, owner introduction, selling points, contact form, and footer
- Reusable data arrays for service cards, project cards, transformations, and selling points
- Contact form success message handled with React state
- Placeholder image boxes ready for real motorcycle client photos

## Run locally

```bash
bun install
bun dev
```

Open the local URL printed by Vite.

## Build

```bash
bun run build
```
